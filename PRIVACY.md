# Privacy Policy

## Canopy Wave Plugin for Dify

**Last Updated:** January 2026

### Data Collection

This plugin collects the following data during operation:

- **User Prompts**: Text messages sent to the model for inference
- **API Credentials**: Your Canopy Wave API key (stored securely by Dify)
- **Model Responses**: Generated text returned from the API
- **Usage Metadata**: Token counts for billing purposes

### Data Usage

- User prompts are transmitted to Canopy Wave API servers for processing
- All data transmission uses HTTPS encryption
- The plugin does not store or log any user prompts or responses locally
- API keys are used solely to authenticate requests to Canopy Wave services

### Third-Party Data Sharing

Data is shared with:

- **Canopy Wave Inc.** - for model inference services
  - Website: https://canopywave.com
  - Privacy Policy: https://canopywave.com/privacy

No other third parties receive data from this plugin.

### Data Retention

This plugin does not retain any user data locally. All data is transmitted directly to Canopy Wave's servers for processing.

For Canopy Wave's data retention and security policies, please refer to their privacy policy at https://canopywave.com/privacy

### Security

- All API communications use TLS encryption
- API keys are stored securely by the Dify platform
- No sensitive data is logged by this plugin

### Contact

For privacy-related inquiries regarding this plugin, please contact:

- **Email**: support@canopywave.io
- **Website**: https://canopywave.io
