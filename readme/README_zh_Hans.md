# Canopy Wave

**作者:** arthur
**版本:** 0.0.5
**类型:** 模型

## 描述

Canopy Wave 插件让 Dify 用户能够通过简单的 API 集成，使用托管在 Canopy Wave 高性能推理平台上的最先进的大语言模型。

该插件利用强大的 AI 模型为您的 Dify 应用提供支持。

## 支持的模型

| 模型                        | 描述                       |
| --------------------------- | -------------------------- |
| deepseek/deepseek-chat-v3.1 | DeepSeek V3.1 对话模型     |
| deepseek/deepseek-chat-v3.2 | DeepSeek V3.2 对话模型     |
| zai/glm-4.7                 | GLM-4.7 对话模型           |
| minimax/minimax-m2.1        | MiniMax M2.1 对话模型      |
| moonshotai/kimi-k2-thinking | Kimi K2 思考模型           |
| moonshotai/kimi-k2.5        | Kimi K2.5 多模态智能体模型 |
| xiaomimimo/mimo-v2-flash    | MIMO V2 Flash 模型         |

## 设置

1. **获取 API 密钥**: 从 [Canopy Wave Cloud](https://cloud.canopywave.io/model-api-key) 获取 API 密钥
2. **安装插件**: 在您的 Dify 工作区中安装此插件
3. **配置**: 在模型提供商设置中添加您的 API 密钥
4. **使用**: 为您的 Dify 应用选择任何 Canopy Wave 模型

## 功能

- ✅ 支持流式和非流式响应
- ✅ 支持多轮对话
- ✅ 支持系统提示词
- ✅ 支持 Temperature 和 Top-P 控制
- ✅ 支持存在惩罚和频率惩罚
