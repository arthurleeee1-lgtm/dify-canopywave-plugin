# Canopy Wave

**作者:** arthur
**バージョン:** 0.0.5
**タイプ:** モデル

## 説明

Canopy Wave プラグインは、Canopy Wave の高性能推論プラットフォームでホストされている最先端の大規模言語モデルへのアクセスを提供します。

このプラグインにより、Dify ユーザーはシンプルな API 統合を通じて強力な AI モデルを活用できます。

## サポートされているモデル

| モデル                      | 説明                                       |
| --------------------------- | ------------------------------------------ |
| deepseek/deepseek-chat-v3.1 | DeepSeek V3.1 チャットモデル               |
| deepseek/deepseek-chat-v3.2 | DeepSeek V3.2 チャットモデル               |
| zai/glm-4.7                 | GLM-4.7 チャットモデル                     |
| minimax/minimax-m2.1        | MiniMax M2.1 チャットモデル                |
| moonshotai/kimi-k2-thinking | Kimi K2 思考モデル                         |
| moonshotai/kimi-k2.5        | Kimi K2.5 マルチモーダルエージェントモデル |
| xiaomimimo/mimo-v2-flash    | MIMO V2 Flash モデル                       |

## セットアップ

1. **APIキーの取得**: [Canopy Wave Cloud](https://cloud.canopywave.io/model-api-key) から API キーを取得します
2. **プラグインのインストール**: Dify ワークスペースにこのプラグインをインストールします
3. **設定**: モデルプロバイダー設定に API キーを追加します
4. **使用**: Dify アプリケーションで任意の Canopy Wave モデルを選択します

## 機能

- ✅ ストリーミングおよび非ストリーミング応答のサポート
- ✅ マルチターン会話のサポート
- ✅ システムプロンプト
- ✅ Temperature と Top-P 制御
- ✅ プレゼンスペナルティとフリークエンシーペナルティ
