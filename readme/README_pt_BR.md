# Canopy Wave

**Autor:** arthur
**Versão:** 0.0.6
**Tipo:** modelo

## Descrição

O plugin Canopy Wave fornece acesso a modelos de linguagem de grande escala de última geração hospedados na plataforma de inferência de alto desempenho da Canopy Wave.

Este plugin permite que os usuários do Dify aproveitem poderosos modelos de IA através de uma simples integração de API.

## Configuração

1. **Obter Chave de API**: Obtenha uma chave de API na [Canopy Wave Cloud](https://cloud.canopywave.io/model-api-key)
2. **Instalar Plugin**: Instale este plugin no seu espaço de trabalho Dify
3. **Configurar**: Adicione sua chave de API nas configurações do provedor de modelo
4. **Usar**: Selecione qualquer modelo Canopy Wave para suas aplicações Dify

## Funcionalidades

- ✅ Suporte a respostas em streaming e não-streaming
- ✅ Suporte a conversação multi-turno
- ✅ Prompts de sistema
- ✅ Controle de Temperature e Top-P
- ✅ Penalidade de presença e frequência
